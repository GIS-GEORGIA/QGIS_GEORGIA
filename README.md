## ქართული

### ჩვენ შესახებ

**QGIS GEORGIA** არის [QGIS](https://qgis.org)-ის დამოუკიდებელი ლოკალური მომხმარებელთა ჯგუფი (Local User Group) საქართველოში — ღია კოდის, მოხალისეობრივი საზოგადოება GIS-ისა და ღია კარტოგრაფიის მოყვარულთათვის.

> [!IMPORTANT]
> QGIS GEORGIA არის საზოგადოების მიერ მართული, **არაოფიციალური** ჯგუფი და არ წარმოადგენს [QGIS.org](https://qgis.org)-ის ან QGIS Project-ის ოფიციალურ ნაწილს.

ეს რეპოზიტორია არის [qgis.ge](https://qgis.ge) საიტის კოდი, რომელიც სერვირდება GitHub Pages-ის მეშვეობით.

### თანამშრომლობა

ვთანამშრომლობთ **[საქართველოს კარტოგრაფთა ასოციაციასთან](https://github.com/Georgian-Cartographers-Association)** — ერთად ვქმნით ახალ ვებ-გვერდებს, აპლიკაციებსა და ღია გეოსივრცულ გადაწყვეტილებებს საქართველოსთვის.

## English

### About

**QGIS GEORGIA** is an independent QGIS Local User Group based in Georgia — an open-source, volunteer-run community for GIS and open-cartography enthusiasts.

> [!IMPORTANT]
> QGIS GEORGIA is a community-run, **unofficial** group and is not an official part of [QGIS.org](https://qgis.org) or the QGIS Project.

This repository holds the source for the [qgis.ge](https://qgis.ge) website, served via GitHub Pages.

### Collaboration

We collaborate with the **[Georgian Cartographers Association](https://github.com/Georgian-Cartographers-Association)** — together we build new websites, applications, and open geospatial solutions for Georgia.

### GIS GEORGIA Projects

Alongside our partnerships, **GIS GEORGIA** independently develops the following open-source projects:

| Project | Description |
|---|---|
| [qgis-plugins-repo](https://github.com/GIS-GEORGIA/qgis-plugins-repo) | A QGIS plugin repository |
| [postgis-manager](https://github.com/GIS-GEORGIA/postgis-manager) | PostGIS Manager — hybrid QGIS plugin & standalone app |
