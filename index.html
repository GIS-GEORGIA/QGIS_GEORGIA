საქართველოს QGIS მომხმარებელთა ღია საზოგადოება Georgia's Open QGIS User Community

QGIS GEORGIA არის QGIS-ის დამოუკიდებელი, არაოფიციალური ლოკალური მომხმარებელთა ჯგუფი (Local User Group) საქართველოში — ღია კოდის, მოხალისეობრივი ინიციატივა GIS-ისა და ღია კარტოგრაფიის მოყვარულთათვის. QGIS GEORGIA is an independent, unofficial QGIS Local User Group based in Georgia — an open-source, volunteer-run initiative for GIS and open-cartography enthusiasts.

ჩვენ შესახებ

About

QGIS-ის ლოკალური ჯგუფი — ღია, დამოუკიდებელი A local QGIS group — open and independent

QGIS GEORGIA აერთიანებს საქართველოში მოღვაწე GIS სპეციალისტებს, კარტოგრაფებს, დეველოპერებსა და სტუდენტებს, რომლებსაც აქვთ საერთო ინტერესი — ღია კოდის გეოსივრცული ინსტრუმენტები და QGIS ეკოსისტემა. QGIS GEORGIA brings together GIS specialists, cartographers, developers and students across Georgia who share an interest in open-source geospatial tools and the QGIS ecosystem.

მნიშვნელოვანია: QGIS GEORGIA არის საზოგადოების მიერ მართული, არაოფიციალური ჯგუფი და არ წარმოადგენს QGIS.org-ს ან QGIS Project-ის ოფიციალურ ნაწილს. Please note: QGIS GEORGIA is a community-run, unofficial group and is not an official part of QGIS.org or the QGIS Project.

Collaboration

ერთად ვქმნით საქართველოს გეოსივრცულ მომავალს Building Georgia's geospatial future, together

ვთანამშრომლობთ საქართველოს კარტოგრაფთა ასოციაციასთან — ერთად ვქმნით ახალ ვებ-გვერდებს, აპლიკაციებსა და ღია გეოსივრცულ გადაწყვეტილებებს. We collaborate with the Georgian Cartographers Association — together we build new websites, applications and open geospatial solutions.

Georgian Cartographers Association
